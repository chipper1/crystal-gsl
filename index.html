Status

GNU Scientific Library (GSL) binding for Crystal

Installation

Add this to your application's shard.yml:

dependencies:
  gsl:
    github: ruivieira/crystal-gsl

System dependencies

Ubuntu < 16.04

  • libgsl0-dev
  • gsl-bin

Ubuntu >= 16.04

  • libgsl-dev
  • gsl-bin

Fedora

  • gsl
  • gsl-devel

Usage

require "gsl"
